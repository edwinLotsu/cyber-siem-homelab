## Wazuh SIEM Deployment (Overview)

Deployed Wazuh SIEM on Ubuntu 24.04 to centralize log collection and analysis across lab machines.

### Components
- Wazuh Manager + Indexer + Dashboard (single-node setup)
- Windows Server 2022 (Domain Controller)
- Windows 10 Client
- Kali Linux (attacker)

### Validation
- Agents successfully connected to SIEM
- Logs ingested from Windows endpoints
- Verified log flow in Wazuh dashboard
